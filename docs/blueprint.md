# **App Name**: MatchMyCofounder Web

## Core Features:

- Landing Page Form: Collect user details like name, email, phone (WhatsApp), and category (Founder, Investor, Student, Freelancer, Professional) through a simple form.
- Playground Section: Provide a 'Playground' section for users to explore and experiment with the platform's features.

## Style Guidelines:

- Primary color: Cyan (#1DAEEF) to maintain brand consistency.
- Background color: Light grayish-blue (#EBF8FF) for a clean look.
- Accent color: Sky Blue (#ADD8E6) for highlights.
- Heading font: 'Poppins' for a modern feel.
- Body font: 'Inter' for readable text.
- Modern layout incorporating a Hero section, animated bento box, sections for top universities/companies, key user stats, testimonials, FAQs, and a global network map.
- Use of accordion elements to organize content.
- Simple, single-page layout focused on the form and playground link.
- Subtle animations for button interactions and form transitions.
- Animated bento box highlighting key platform features and data.
- Animated counters for displaying user and startup statistics.
